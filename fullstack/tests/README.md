# FizzBuzz

Print every number from 1 to 100 (both included) on a new line. Numbers which are multiple of 3, print “Fizz” instead of a number. For the numbers which are multiples of 5, print “Buzz” instead of a number. For the number which is multiple of both 3 and 5, print “FizzBuzz” instead of numbers.

# Api countries

## Backend
- Design an endpoint `/countries` that send all data contained in [countries.json](./countries.json) file
- Design an endpoint `/countries?region=xxxx` that send data contained in [countries.json](./countries.json) file **filter on region field**

## Frontend
In a frontend application, based on the second endpoint display each Africa countries in a card :
 - Name of country (common name)
 - Population
 - Languages
 - Capital (could be several, have look on South Africa)
 - Flag in png
